## 📌 Project Overview

Customer churn prediction is one of the most important applications of Machine Learning in business analytics. Retaining existing customers is significantly more cost-effective than acquiring new ones, making churn prediction a valuable decision-support tool.

This project develops and compares multiple machine learning classification models to predict whether a customer is likely to churn based on demographic, subscription, and behavioral information.

The project follows a complete machine learning workflow:

- 📥 Data Loading
- 📊 Exploratory Data Analysis (EDA)
- 🧹 Data Preprocessing
- 🤖 Training Multiple Machine Learning Models
- 📈 Model Evaluation & Comparison
- 📌 Business Insights

---

# 🎯 Objectives

- Analyze customer behavior patterns
- Identify important factors influencing churn
- Train multiple machine learning classifiers
- Compare model performance using standard evaluation metrics
- Select the best-performing model for churn prediction

---

# 📂 Dataset

The dataset contains customer information such as:

- Customer Age
- Gender
- Subscription Type
- Account Age
- Viewing Hours
- Support Tickets
- Payment Method
- Customer Preferences
- Other customer-related features

### Target Variable

```
Churn
```

- **1 → Customer Churned**
- **0 → Customer Retained**
